# Eduon 📚

A modern and comprehensive educational platform built with [Flutter](https://flutter.dev/).

## 📖 About the Project
Eduon is a cross-platform learning application designed to provide users with a seamless educational experience. Built with Flutter & Dart, the project integrates course management, structured learning paths, user authentication, and AI services.

## ✨ Key Features

### 📚 Courses
- 🔍 **Course Search** — Instantly search and find any course by name or topic
- 🗂️ **Course Categories** — Browse courses by category:
  - 💻 Tech · 🎨 Design · 💼 Business · 🧠 Soft Skills · 🎬 Video Editing
- 🎬 **Playlist Viewer** — Watch any playlist freely with full video control
- 📊 **Progress Tracking** — Track course watch percentage and remaining content in real-time

### 🗺️ Learning Paths
- Structured roadmaps composed of multiple courses for complete skill tracks
- Covers popular tracks for well-known career paths

### 🎯 Activities
- Explore top educational platforms and communities directly from the app
- Supported organizations:
  - 🔵 IEEE Student Branch
  - 🟠 Fares Toson Academy
  - 🔷 Microsoft Student Club
  - 🟡 Google Developer Groups
  - 🟢 Oi HUB & Oi ROV
  - 🔴 ICPC OBOUR
  - 🤖 ROBOTICS

### ⚙️ General
- 🔐 **User Authentication** — Secure login and registration with Firebase
- 👤 **Profile Management** — Image uploading and permissions management
- 🤖 **AI Service Integration** — Built-in AI features with custom model configurations
- 🌙 **Dark & Light Mode** — Full theme support for better user experience
- 🔔 **Push Notifications** — Real-time notifications to keep users engaged
- 📱 **Cross-Platform** — Fully supported on Android & iOS

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- Dart SDK
- VS Code or Android Studio

### Installation
1. Clone the repository:
```bash
   git clone https://github.com/MohamedRefky/Eduon.git
```

2. Navigate to the project directory:
```bash
   cd Eduon
```

3. Install dependencies:
```bash
   flutter pub get
```

4. Run the application:
```bash
   flutter run
```
