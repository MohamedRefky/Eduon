# Eduon 📚

A modern and comprehensive educational platform built with [Flutter](https://flutter.dev/).

## 📖 About the Project
Eduon is a cross-platform learning application designed to provide users with a seamless educational experience. Built with Flutter & Dart, the project integrates course management, structured learning paths, user authentication, and AI services.

## ✨ Key Features
- 🔐 **User Authentication** — Secure login and registration with Firebase
- 👤 **Profile Management** — Image uploading and permissions management
- 📚 **Course Management** — Explore and manage learning paths via API integration
- 🤖 **AI Service Integration** — Built-in AI features with custom model configurations
- 📱 **Cross-Platform** — Fully supported on Android & iOS

## 🚀 Getting Started

### Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- Dart SDK
- VS Code or Android Studio

### Installation
1. Clone the repository:
```bash
   git clone https://github.com/MohamedRefky/Eduon.git
```

2. Navigate to the project directory:
```bash
   cd Eduon
```

3. Install dependencies:
```bash
   flutter pub get
```

4. Run the application:
```bash
   flutter run
```
